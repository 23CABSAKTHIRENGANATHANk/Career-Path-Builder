## 🐛 IF SOMETHING'S WRONG

### "No Report Data Found"
→ Go home, fill form again, submit

### "Dashboard not showing"
→ Make sure backend is running (port 5000)

### "API connection error"
→ Check: `curl http://127.0.0.1:5000/api/health`

### "PDF download not working"
→ Try Print to PDF (Ctrl+P) instead

### "Page looks wrong"
→ Clear cache: Ctrl+Shift+Del (or Cmd+Shift+Del)
